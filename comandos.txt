# Instruções para criação da imagem

# Abra o terminal ou prompt de comando e navegue até a pasta onde você colocou os arquivos do iac_terraform_docker_googlebigquery (não use espaço ou acento em nome de pasta)

# Execute o comando abaixo para criar a imagem Docker

docker build -t image-modelagem-iac:iac_project .

# Execute o comando abaixo para criar o container Docker

docker run -dit --name my-dw-iac -v C:/Users/carlo/Downloads/iac_terraform_docker_googlebigquery/iac_bigquery:/iac_bigquery image-modelagem-iac:iac_project /bin/bash

NOTA: No Windows você deve substituir ./Lab1 pelo caminho completo da pasta, por exemplo: C:\meusdocumentos\Lab1\iac_bigquery

# Verifique as versões

terraform version
gcloud version
